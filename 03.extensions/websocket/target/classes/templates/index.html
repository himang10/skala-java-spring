클라이언트가 force-disconnect 프레임을 보내면 서버가 현재 세션을 닫고, 페이지는 close 이벤트를 받은 뒤 약 3초 후 자동 재연결합니다.

i흐름 요약
1) 연결 버튼은 /ws/connect?clientId={id}로 WebSocket 연결을 생성합니다.
2) 이후 통신은 HTTP가 아니라 JSON 텍스트 프레임 송수신입니다.
2-1) 오른쪽 패널에서 서버가 발송한 이벤트를 수신 순서대로 확인합니다.
3) 강제 종료도 REST 호출 없이 force-disconnect 프레임으로 요청합니다.
4) 세션이 닫히면 사용자가 수동 종료하지 않은 경우 자동 재연결합니다.
