# 개요
본 프로젝트는 spring boot 기반으로 SSE 사용 방법을 위한 Server와 thymeleaf 기반 교육용 프로그램을 만드는 것이다.


# SPEC
- JAVA Version: 21
- spring boot : 3.x 
- maven 과 gradle 버전 모두 지원

# 규칙
다음이 명확하게 교육이 되도록 간단한 코드를 작성해라
각 코드는 아래의 단계별로 이해할 수 있도록 구분해서 테스트 가능하도록 한다.

- SSE 연결 과정
- 연결 후 정상 통신 과정
- 연결이 끊어질때 재 연결 과정

불필요한 코드는 빼록 명확하게 매커니즘을 이해하는데 중점을 둔 코드를 작성해라
사용자 화면 역시 위의 과정을 테스트하고 테스트 결과를 볼수 있도록 해라
로그는 프로토콜 과정과 메시지 구조가 명확하게 보이도록 로그를 작성해라
각 소스코드는 클래스, 메소드에 대한 Javadoc 기반 comment를 작성해라
중요 코드에 대해서는 한줄의 설명 comment를 작성해라
